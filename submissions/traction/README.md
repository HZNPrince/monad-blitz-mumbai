# Traction - AI-Powered Distribution Agent for Monad Blitz

Transform any website into a marketing powerhouse with AI-generated campaigns and Monad blockchain-verified traction.

## 🎯 Overview

**Traction** is an autonomous distribution agent that:
1. **Analyzes** your website in 15-25 seconds
2. **Extracts** brand DNA (identity, features, benefits, audiences)
3. **Generates** 4 professional marketing campaigns
4. **Reviews** creatives in Blitz Mode (Vibiz.ai-style UI)
5. **Tracks** performance on Monad blockchain

## ✨ Key Features

### Frontend
- **Vibiz.ai-style Brand DNA Editor** - Customize brand profiles interactively
- **Premium Blitz Mode** - Stacked card interface for rapid creative review
- **Professional UI** - 2026 trending design (minimalist, motion graphics)
- **Full Workflow** - URL → Analysis → DNA → Blitz Mode → Approve/Reject → Draft

### Backend
- **Firecrawl Integration** - Intelligent website scraping
- **AI Analysis** - Product DNA extraction using Cloudflare LLMs
- **Creative Generation** - Premium image prompts with 2026 design trends
- **Activity Tracking** - Full audit trail of all actions

### Blockchain
- **Smart Contracts** - TractionCampaignFactory on Monad testnet
- **Performance Rewards** - Traction-verified compensation
- **NFT Campaigns** - ERC721 for each campaign
- **Oracle Integration** - Real metrics verification

## 📖 How It Works

1. **Submit Website URL** → Traction analyzes your site
2. **Review Brand DNA** → Interactive customization of brand profile
3. **Launch Blitz Mode** → Review 4 AI-generated campaigns
4. **Approve/Reject** → Use keyboard shortcuts (← →)
5. **Edit Drafts** → Customize copy and visual layers
6. **Deploy Campaign** → Lock funds on Monad smart contract

## ⚡ Quick Start

```bash
# Clone and install
git clone https://github.com/HZNPrince/traction.git
cd traction
npm install

# Set up environment
cp .env.example .env.local
# Add your API keys

# Run locally
npm run dev
# Visit http://localhost:3000
```
